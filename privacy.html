基本方針

Overview

Scrappy（以下「本アプリ」）は、ユーザーのプライバシーを最優先に設計されています。本アプリは一切のデータを外部サーバーに送信しません。すべての処理はお使いの端末内で完結します。

Scrappy (the "App") is designed with your privacy as the top priority. The App does not send any data to external servers. All processing is performed entirely on your device.

最終更新日: 2026年5月8日

Last updated: May 8, 2026

1. 収集する情報

1. Information Accessed

本アプリは、以下の情報に端末上でのみアクセスします。外部への送信・収集は行いません。

The App accesses the following information on-device only. No data is sent or collected externally.

データ 用途 外部送信
スクリーンショット画像 OCRテキスト認識による自動分類 なし
写真メタデータ（撮影日時） 古いスクショの検出・時間帯による分類 なし
ユーザー設定（分類ルール・日数） アプリの動作カスタマイズ なし
Data Purpose Sent Externally
Screenshot images OCR text recognition for auto-classification No
Photo metadata (capture date) Detecting old screenshots / time-based sorting No
User settings (rules, days) App behavior customization No

2. 写真ライブラリへのアクセス

2. Photo Library Access

本アプリは、スクリーンショットの読み取りおよび削除のため、iOS標準の写真ライブラリアクセス権限を使用します。

The App uses standard iOS Photo Library access to read and delete screenshots.

  • アクセスは初回起動時にユーザーの明示的な許可を得て行います
  • 写真データは端末内でのみ処理され、コピー・キャッシュ・アップロードは行いません
  • 削除操作は必ず確認ダイアログを表示し、誤操作を防止します
  • 削除されたスクショはiOS標準の「最近削除した項目」に30日間保持されます
  • Access is granted only with your explicit permission on first launch
  • Photo data is processed on-device only — no copying, caching, or uploading
  • Deletion always requires a confirmation dialog to prevent accidental loss
  • Deleted screenshots are kept in iOS "Recently Deleted" for 30 days

3. データの保存

3. Data Storage

本アプリが保存するデータは以下のみです。すべて端末内のUserDefaultsに保存され、外部と共有されることはありません。

The App stores only the following data in on-device UserDefaults. None of it is shared externally.

  • 整理提案の日数設定（デフォルト: 30日）
  • カスタム分類ルール（キーワードとカテゴリの組み合わせ）
  • Cleanup threshold setting (default: 30 days)
  • Custom classification rules (keyword-category pairs)

分類結果（カテゴリ情報）はメモリ上のみで管理され、永続的に保存されません。

Classification results (category assignments) are held in memory only and are not persisted.

4. Third-Party Services & SDKs

  • Third-party analytics tools (Firebase, Amplitude, etc.)

  • External APIs or server communication

5. Apple Intelligence

Apple Intelligence対応端末では、Appleが提供するon-device画像理解機能を利用して高精度な分類を行います。この処理もすべて端末内で完結し、Appleのサーバーにデータが送信されることはありません。

On Apple Intelligence-capable devices, the App uses Apple's on-device image understanding for higher-accuracy classification. This processing is also performed entirely on-device — no data is sent to Apple's servers.

6. 子供のプライバシー

6. Children's Privacy

本アプリは13歳未満の子供から個人情報を意図的に収集することはありません。そもそもいかなるユーザーからも個人情報を収集しません。

The App does not intentionally collect personal information from children under 13. In fact, it does not collect personal information from any user.
